# Changelog

All notable changes to the Autonomous Business Ops Copilot will be documented here.

## [1.0.0] - 2026-07-10

### Added
- Scaffolding, database schema structures, and FastAPI gateways (Phase 2 & 3).
- Hybrid retrieval pipelines combining BM25, ChromaDB vector dense matching, and Neo4j GraphRAG (Phase 4).
- Multi-agent orchestration workflows using LangGraph and Planner DAG routers (Phase 5).
- Multi-page Streamlit dashboards workspace (Phase 6).
- Ragas/DeepEval evaluations, throughput checks load runners, and regression reports scripts (Phase 7).
